### 一、安装依赖

```bash
sudo apt update
sudo apt install certbot python3-certbot-nginx -y
```

### 二、确保域名已解析

```bash
ping example.com
curl http://example.com
```

### 三、申请证书

```bash
sudo certbot --nginx -d example.com -d www.example.com
```

### 四、验证证书安装

1. 存放目录

   ```swift
   /etc/letsencrypt/live/example.com/
   ```

2. 确认

   ```bash
   sudo nginx -t
   sudo systemctl reload nginx
   ```
